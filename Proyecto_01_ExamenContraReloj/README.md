# Examen Contra Reloj Unidad 2 - Corte 1
**Objetivo del proyecto**

El objetivo es evaluar la precisión técnica en la sintaxis de PHP y la capacidad de representar datos sin utilizar comportamientos o métodos.

**Problema que resuelve**

El proyecto permite organizar información de paquetes y sensores mediante clases en PHP.

+ Logística: registrar datos de paquetes como código, peso y fragilidad.
+ Monitoreo: guardar información de sensores y su última lectura.

**Tecnologías utilizadas**
+ PHP
+ Apache / XAMPP
+ Visual Studio Code

**Conceptos aplicados**
+ Clases y objetos
+ Encapsulamiento (public y private)
+ Tipado estricto
+ Uso de require_once
