# Gestión de Bitácoras en Archivos de Texto (PHP)
**Objetivo del proyecto**

Aprender a guardar y leer información en archivos de texto utilizando PHP.

**Problema que resuelve**

Permite registrar actividades de forma digital y conservar un historial sin necesidad de utilizar una base de datos.

**Tecnologías utilizadas**
- PHP
- HTML
- XAMPP
- Visual Studio Code

**Conceptos aplicados**
- Manejo de archivos de texto
- Persistencia de datos
- Modo append
- Validación de formularios

**Instrucciones de ejecución**
1. Guardar los archivos en la carpeta htdocs.
2. Iniciar Apache en XAMPP.
3. Abrir localhost/bitacora/index.php.
4. Registrar actividades mediante el formulario.

**Reflexión personal**
- ¿Qué aprendí? Aprendí que se pueden crear sistemas para guardar datos reales sin usar bases de datos complejas, usando herramientas sencillas de PHP para escribir y leer archivos de texto plano. También aprendí a usar etiquetas como *pre* o listas ordenadas para que el texto plano se acomode bonito en la pantalla.
 
- ¿Qué fue difícil? Lo más difícil fue asegurarme de que cada vez que guardara una tarea nueva, esta se escribiera en un renglón abajo sin encimarse ni borrar el texto que ya estaba guardado antes. También tuve que cuidar que no se guardaran registros en blanco si se presionaba el botón por accidente.

- ¿Qué mejoraría? Mejoraría el sistema agregando un botón para poder borrar filas específicas de la bitácora o una opción para filtrar y buscar las actividades por fecha o por el nombre del responsable, para no tener que leer toda la lista completa cuando crezca mucho.
