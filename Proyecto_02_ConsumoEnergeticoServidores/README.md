# Analizador de Consumo Energético en Servidores

**Objetivo del proyecto**

Desarrollar una aplicación web que calcule el consumo de energía de un servidor utilizando programación orientada a objetos y métodos numéricos.

**Problema que resuelve**

Permite calcular de forma automática la energía consumida por un servidor, facilitando el análisis de costos y consumo energético.

**Tecnologías utilizadas**
- PHP
- HTML5
- CSS3
- Apache / XAMPP
- Visual Studio Code

**Conceptos aplicados**
- Programación orientada a objetos
- Métodos y clases
- Encapsulamiento
- Namespaces
- Manejo de excepciones (try-catch)

**Instrucciones de ejecución**

1. Copiar la carpeta del proyecto en htdocs.
2. Iniciar Apache en XAMPP.
3. Abrir el proyecto desde localhost.
4. Ingresar los datos y presionar Calcular.

**Reflexión personal**
- ¿Qué aprendí? Aprendí a conectar la lógica de una fórmula matemática (la regla del Trapecio) con una página web dinámica mediante métodos en PHP. También aprendí a usar namespace para ordenar mejor mis proyectos y a implementar bloques try-catch para controlar los errores de entrada del usuario y hacer que el sistema sea más seguro.

- ¿Qué fue difícil? Lo más difícil fue armar el ciclo for para que sumara todo correctamente sin equivocarme en los signos o las letras. Al principio también me dio dolor de cabeza entender cómo mandar los datos que la gente escribe en la pantalla hacia el código que hace los cálculos.

- ¿Qué mejoraría? Me gustaría aprender a usar más estilos CSS para que la página web se vea más bonita y profesional. También quiero practicar más ejercicios parecidos para agarrarle más velocidad y entender mejor cómo procesar fórmulas rápido.
