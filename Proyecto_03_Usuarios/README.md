# ABPj: Sistema de Usuarios con Herencia en Python (un cambio de lenguaje)

**Objetivo del proyecto**

Aprender a utilizar la herencia y el polimorfismo en Python mediante la creación de distintos tipos de usuarios.

**Problema que resuelve**

Permite administrar diferentes tipos de usuarios sin repetir código, compartiendo datos comunes y personalizando sus permisos.

**Tecnologías utilizadas**
- Python 3.x
- Visual Studio Code

**Conceptos aplicados**
- Herencia
- Polimorfismo
- Sobrescritura de métodos
- Uso de super()

**Instrucciones de ejecución**
1. Guardar todos los archivos en una misma carpeta.
2. Abrir la terminal o Visual Studio Code.
3. Ejecutar el comando python main.py.
4. Utilizar las opciones del menú.
   
**Reflexión personal**
- ¿Qué aprendí? Aprendí a usar la herencia en Python, lo cual ayuda muchísimo a ahorrar tiempo y no escribir el mismo código una y otra vez. También aprendí a usar super() para conectar los constructores y a cambiar el comportamiento de una función usando la sobrescritura.

- ¿Qué fue difícil? Lo más difícil fue entender cómo pasarle los parámetros a la clase padre con super() y estructurar el menú interactivo para que recorriera la lista de usuarios usando polimorfismo sin que saltaran errores en la terminal.

- ¿Qué mejoraría? Me gustaría mejorar el programa agregando una base de datos real para que los usuarios se queden guardados permanentemente y no se borren cada vez que cierro la terminal.
